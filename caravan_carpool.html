Caravan Carpool

Summer 2020

Technologies

React Native/Expo, Firebase, GCP

Role

Full Stack Mobile Developer Intern

Overview

Caravan is an iOS app that connects Colorado skiers and snowboarders looking to carpool to the resorts. Once a user creates and verifies their account, they can create a new ride or request a ride from a list of existing rides. They can keep track of rides in the “My Rides” page and update them as needed. Users can also add information to their profile such as a picture, nickname, and vehicle information.

Responsibilities

  • Launched Expo React Native iOS carpool app for skiers in Denver with a team of six engineers
  • Led pair programming sprints and implementation of multiple Expo APIs in full stack agile development
  • Removed database redundancy by over 67% by streamlining database structure and leading refactorization
  • Wrote automated database maintenance script that runs twice a week using a cron job with Google Cloud Scheduler
  • Defined product strategy by writing functional requirements and user workflows prior to development and release
